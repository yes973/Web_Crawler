## 참고하세요
웹사이트의 소유자가 허용하지 않는 웹 크롤링은 합법과 불법의 경계선상에 있습니다. 
현행법상 웹 크롤링의 결과를 개인적으로 갖고 있는 것은 괜찮지만
저작권자의 명시적 허락 없이 타 사이트에 업로드하거나 상업적 목적으로 악용하는 경우 법적인 문제가 발생할 수 있습니다.
